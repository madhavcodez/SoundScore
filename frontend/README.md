# SoundScore Frontend - 2025 Beta v0.01

This frontend is the legacy web client for **SoundScore (2025 beta v0.01)**.

It is an open-source fun web app developed in 2025 and retained as an archival client while the next production app is in progress.

## Available Scripts

In the `frontend` directory, you can run:

### `npm start`

Starts the development server.

### `npm test`

Runs the test runner in watch mode.

### `npm run build`

Builds a release bundle into the `build` folder.
